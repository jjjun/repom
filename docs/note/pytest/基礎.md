
## テストファイルとテスト関数の違い

- テストファイル: テストファイルは、テストコードを含む Python ファイルです。通常、ファイル名は test_ で始まるか、_test で終わるように命名されます。例えば、test_note1.py や note1_test.py などです。
- テスト関数: テスト関数は、テストファイル内に定義された関数で、通常は test_ で始まる名前を持ちます。例えば、test_sample1 や test_example などです。

pytest を実行すると、自動的に `test_` を含むテストファイルを実行する。ただ、`test_` を含むファイルじゃ無いとテスト出来ないのかというと、そう言う事でも無くて、`pytest 'docs/pytest/sample/note1.py'` の様に、直接テストファイルを指定すれば実行できる。
